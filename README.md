# PAY.POS Local ECR

### Requirments:
- PAY.POS version 1.1.18 or higher
- Android terminals only (Android softpos, Sunmi hardpos, and PAX hardpos)
- Your network needs to be able to send UDP (port 8889) & TCP (port 8888) messages
